_Delete Project_

Warning! Deletion of a project is permanent action!

Deleting a project will have the following consiquences:

  • Access to this project's samples will be removed for all users of this project.
  • Samples which have been copied to other projects will not be affected. They will still be available in their other projects.
  • Reference files associated with this project will no longer be accessable.
  • Associated projects will no longer have access to this project's samples.
  • Analyses shared with this project will no longer be shared with this project's users. Submitted analyses will still be avilable to the submitting users in their analysis list
  • Information about NCBI submissions from this project will no longer be accessable. Data which has been uploaded to NCBI will still be available on NCBI's sites.
